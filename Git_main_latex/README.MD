

# README — Как пользоваться шаблоном для домашних заданий

Этот шаблон предназначен для удобного написания домашних заданий по математике.
Он автоматически нумерует задачи, поддерживает два типа подпунктов, содержит команды для математических обозначений и окружения для решений.

Документ уже настроен, поэтому писать нужно **только внутри документа**.

```
\begin{document}
...
\end{document}
```

---

# 1. Новая задача

## Команда

```
\task
```

Создаёт новую задачу с автоматической нумерацией.

Пример:

```
\task
Найти предел функции.
```

Результат:

```
Задача 1.
Найти предел функции.
```

---

## Задача с названием

Можно добавить название:

```
\task[Предел функции]
```

Результат:

```
Задача 1. Предел функции
```

---

# 2. Подпункты внутри задачи

Есть **два независимых типа подпунктов**.

Они **автоматически обнуляются при новой задаче**.

---

# 2.1 Числовые подпункты `(1) (2) (3)`

## Команда

```
\subtasknum
```

Пример:

```
\subtasknum
Найти предел.

\subtasknum
Доказать сходимость.
```

Результат:

```
(1) Найти предел
(2) Доказать сходимость
```

---

# 2.2 Буквенные подпункты `а) б) в)`

## Команда

```
\subtaskabc
```

Пример:

```
\subtaskabc
Найти производную.

\subtaskabc
Найти вторую производную.
```

Результат:

```
а) Найти производную
б) Найти вторую производную
```

---

# 2.3 Можно смешивать подпункты

```
\subtasknum
...

\subtasknum
...

\subtaskabc
...

\subtaskabc
...
```

Результат:

```
(1)
(2)
а)
б)
```

---

# 3. Оформление решения

## Команда

```
\begin{solution}
...
\end{solution}
```

Пример:

```
\begin{solution}
Используем определение предела.
\end{solution}
```

Результат:

```
Решение:
Используем определение предела.
```

---

# 4. Оформление ответа

## Команда

```
\begin{answer}
...
\end{answer}
```

Пример:

```
\begin{answer}
1
\end{answer}
```

Результат:

```
Ответ:
1
```

---

# 5. Математические множества

Используются **в математическом режиме** `$...$`.

| команда | результат         |
| ------- | ----------------- |
| `$\N$`  | натуральные числа |
| `$\Z$`  | целые             |
| `$\Q$`  | рациональные      |
| `$\R$`  | вещественные      |
| `$\CC$` | комплексные       |
| `$\F$`  | поле              |

Пример:

```
$f:\R\to\R$
```

---

# 6. Полезные математические операторы

| команда | результат           |
| ------- | ------------------- |
| `\rk`   | ранг                |
| `\tr`   | след                |
| `\grad` | градиент            |
| `\spec` | спектр              |
| `\ord`  | порядок             |
| `\Mat`  | пространство матриц |

Пример:

```
$\rk A = 3$
```

---

# 7. Частные производные

Команда:

```
\p{f}{x}
```

Пример:

```
$\p{f}{x}$
```

Результат:

```
∂f / ∂x
```

---

# 8. Пределы

Команда:

```
\Lim{x}{0}
```

Пример:

```
$\Lim{x}{0} f(x)$
```

Результат:

```
lim_{x→0} f(x)
```

---

# 9. Суммы

Команда:

```
\Sum{i=1}{n}
```

Пример:

```
$\Sum{i=1}{n} i$
```

---

# 10. Красивые системы уравнений

Для систем используется окружение `cases`.

## Пример

```
\[
\begin{cases}
x+y=4 \\
x-y=2
\end{cases}
\]
```

Результат:

```
{ x+y=4
{ x-y=2
```

---

# 10.1 Система с функцией

```
\[
f(x)=
\begin{cases}
x^2, & x\ge0 \\
-x, & x<0
\end{cases}
\]
```

Используется для кусочных функций.

---

# 11. Перенос строк в системах

В системах используется:

```
\\
```

Пример:

```
\begin{cases}
x+y=1 \\
x-y=3
\end{cases}
```

---

# 12. Пример полноценной задачи

```
\task[Система уравнений]

\subtasknum
Решить систему

\[
\begin{cases}
x+y=4 \\
x-y=2
\end{cases}
\]

\begin{solution}

Складываем уравнения

\[
2x=6
\]

\[
x=3
\]

Тогда

\[
y=1
\]

\end{solution}

\begin{answer}
$x=3,\;y=1$
\end{answer}
```

---

# 13. Минимальный шаблон для ДЗ

Обычно задача пишется так:

```
\task

\subtasknum
...

\subtasknum
...

\begin{solution}
...
\end{solution}

\begin{answer}
...
\end{answer}
```

---

# 14. Полезные рекомендации

1. Формулы писать в

```
$
$
```

или

```
\[
\]
```

2. Каждую новую задачу начинать с

```
\task
```

3. Подпункты использовать только **внутри задачи**.
