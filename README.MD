# LaTeX-шаблон для домашних работ



## 1. Обзор

Данный шаблон предназначен для оформления математических домашних работ.
Он включает:

* автоматическую нумерацию задач
* структурированные окружения решения и ответа
* предопределённые математические операторы и множества
* расширения для линейной алгебры
* встроенную поддержку 2D и 3D графиков через `pgfplots`
* опциональную поддержку символьных вычислений через SymPy


## 2. Требования

### Дистрибутив TeX

Рекомендуется установка **TeX Live Full**.

Проверка:

```bash
latexmk -v
```

### Редактор

VS Code с установленным расширением:

* LaTeX Workshop

Сборка:

* `Ctrl + Alt + B` — собрать документ
* `Ctrl + Alt + V` — открыть PDF


## 3. Структура документа

### Заголовочный блок

```latex
\title{Название курса \\ \large Номер работы}
\author{Имя}
\affil{Группа}
\date{\today}
```

Вывод осуществляется командой:

```latex
\maketitle
```


## 4. Окружение задачи

### Команда

```latex
\task
```

Создаёт новую задачу с автоматической нумерацией.

Пример:

```latex
\task
```

Вывод:

```
Задача 1.
```


### Задача с названием

```latex
\task[Название]
```

Пример:

```latex
\task[Теорема о ранге]
```

Вывод:

```
Задача 2. Теорема о ранге
```


### Ссылка на задачу

```latex
\task
\label{t:example}

См. задачу \ref{t:example}.
```

## 10. Расширенный модуль линейной алгебры

### 10.1 Матрицы

#### Матрица в круглых скобках

```latex
\mat{a & b \\ c & d}
```

Создаёт матрицу в круглых скобках.

Пример:

```latex
\[
\mat{1 & 2 \\ 3 & 4}
\]
```

---

#### Матрица в квадратных скобках

```latex
\bmat{a & b \\ c & d}
```

Создаёт матрицу в квадратных скобках.

---

#### Определитель

```latex
\detmat{a & b \\ c & d}
```

Создаёт матрицу в вертикальных чертах для записи определителя.

---

#### Вектор-столбец

```latex
\vect{a \\ b \\ c}
```

Создаёт вектор-столбец в круглых скобках.

---

#### Вектор в квадратных скобках

```latex
\bvect{a \\ b \\ c}
```

Создаёт вектор-столбец в квадратных скобках.

---

### 10.2 Нормы и скалярные произведения

#### Модуль или абсолютное значение

```latex
\abs{x}
```

Записывает выражение в одиночных вертикальных чертах.

---

#### Норма

```latex
\norm{v}
```

Записывает норму с двойными вертикальными чертами.

---

#### Скалярное произведение

```latex
\scalar{x}{y}
```

Записывает скалярное произведение двух векторов.

---

### 10.3 Транспонирование и обратная матрица

#### Транспонирование

```latex
A\T
```

Записывает транспонированную матрицу.

---

#### Обратная матрица

```latex
A\inv
```

Записывает обратную матрицу.

---

### 10.4 Пространства и операторы

#### Линейная оболочка

```latex
\Span\set{v_1, v_2}
```

Записывает линейную оболочку векторов.

---

#### Ядро

```latex
\Ker A
```

Записывает ядро линейного отображения.

---

#### Образ

```latex
\Img A
```

Записывает образ линейного отображения.

---

#### Размерность

```latex
\dimn V
```

Записывает размерность пространства.

---

#### Пространство столбцов

```latex
\Col A
```

---

#### Пространство строк

```latex
\Row A
```

---

#### Диагональная матрица

```latex
\diag(a_1, a_2, a_3)
```

---

### 10.5 Системы линейных уравнений

```latex
\system{
x + y &= 1 \\
2x - y &= 0
}
```

Формирует систему уравнений с фигурной скобкой слева и автоматическим выравниванием.

---

### 10.6 Расширенная матрица

#### Универсальная запись через вертикальную черту

```latex
\augmat
{a_{11} & a_{12} \\ a_{21} & a_{22}}
{b_1 \\ b_2}
```

Создаёт расширенную матрицу вида ( [A \mid b] ).

Пример:

```latex
\[
\augmat
{1 & 2 \\ 3 & 4}
{5 \\ 6}
\]
```

---

### 10.7 Операции метода Гаусса

#### Операция над строками

```latex
\rowop{R_2 \leftarrow R_2 - 3R_1}
```

Добавляет подпись операции над строками над стрелкой.

Использование:

```latex
A \rowop{R_2 \leftarrow R_2 - 3R_1} B
```

---

#### Переход к ступенчатому виду

```latex
\toREF
```

Используется для обозначения перехода к ступенчатому виду.

Пример:

```latex
A \toREF B
```

---

#### Переход к приведённому ступенчатому виду

```latex
\toRREF
```

Используется для обозначения перехода к приведённому ступенчатому виду.

Пример:

```latex
A \toRREF B
```

---

### 10.8 Типовой пример применения

```latex
\[
\augmat
{1 & 2 \\ 3 & 4}
{5 \\ 6}
\rowop{R_2 \leftarrow R_2 - 3R_1}
\augmat
{1 & 2 \\ 0 & -2}
{5 \\ -9}
\toRREF
\augmat
{1 & 0 \\ 0 & 1}
{? \\ ?}
\]
```



## 11. Модуль построения графиков

Шаблон включает:

* `tikz`
* `pgfplots`
* преднастроенный стиль для 2D
* преднастроенный стиль для 3D


### 2D график

```latex
\begin{tikzpicture}
\begin{axis}
\addplot {x^2};
\end{axis}
\end{tikzpicture}
```


### Несколько функций

```latex
\addplot {x^2};
\addplot {x};
\legend{$x^2$, $x$}
```


### 3D поверхность

```latex
\begin{tikzpicture}
\begin{axis}[3dstyle]
\addplot3[surf, domain=-2:2]
{x^2 + y^2};
\end{axis}
\end{tikzpicture}
```


### Изменение угла обзора

```latex
\begin{axis}[3dstyle, view={45}{40}]
```
